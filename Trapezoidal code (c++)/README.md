# Trapezoidal Rule Numerical Integration

## Description
This project implements the Trapezoidal Rule in C++ to compute the definite integral of a function numerically.

The function used in this program is:
f(x) = 1 / (1 + x^2)

## Goal
Approximate the value of a definite integral using numerical integration methods and understand error behavior with different subintervals.

## Features
- User input for integration limits (a, b)
- User input for number of subintervals (n)
- Implementation of Trapezoidal Rule
- Numerical approximation of the integral
- Uses arrays to store function values

## Formula Used
Integral ≈ (h/2) [y0 + yn + 2(y1 + y2 + ... + y(n-1))]

## Technology
C++ (Standard Library)
