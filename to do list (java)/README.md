# To-Do List Application

## Description
This project implements a simple To-Do List system using a model-based structure with JList and JModel.

## Goal
Manage a list of items using basic operations such as adding, removing, and updating the list state.

## Structure
- JModel: Represents the main data model of the application
- JList: Stores and manages the list of items

## Features
- Add item to the list (Add)
- Remove item from the list (Remove)
- Mark item as done (Mark as Done)
- Manage and update list through JModel

## Operations Logic
- Add → Insert item into JList
- Remove → Delete item from JList
- Mark as Done → Update item state inside JModel / JList

## Technology
Java (OOP + Custom Model/List Structure)
